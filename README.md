Node.js / Express.js backend для онлайн-игры «Тетрис» с системой аккаунтов, сохранением рекордов и таблицей лидеров.  

Разработан REST API для:  
• регистрации и авторизации пользователей;  
• безопасного хранения аккаунтов и JWT-аутентификации;  
• сохранения рекордов игроков в базе данных;  
• получения глобального leaderboard;  
• обновления статистики и профиля пользователя.  

Технологии:  
• Node.js  
• Express.js  
• PostgreSQL  
• JWT Authentication  
• bcrypt  
• CORS  
• REST API  

Функционал:  
• создание аккаунта и вход в систему;  
• сохранение лучших результатов игрока;  
• вывод топа игроков по очкам;  
• защита API и валидация запросов;  
• клиент-серверное взаимодействие через fetch/Axios.  

Backend спроектирован по принципам реальных production-приложений: разделение роутов, middleware, контроллеров и работы с базой данных.
